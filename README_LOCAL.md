
# La Machine à Écrire - Version Locale Python

Cette version a été créée pour résoudre vos problèmes de portabilité et de configuration web.

## Comment l'utiliser dans PyCharm :
1. Créez un fichier `machine.py` et collez le code fourni.
2. Si vous avez une erreur sur `tkinter` : 
   - Windows : Relancez l'installeur Python et cochez "tcl/tk".
   - PyCharm : Vérifiez que l'interpréteur est bien celui de votre système.
3. Lancez le script (`Right Click` -> `Run machine`).

## Pourquoi c'est mieux pour vous :
- **Zéro serveur** : Double-cliquez sur le fichier pour écrire.
- **Fichiers .mac** : Vous pouvez ouvrir vos scripts créés sur la version web.
- **Vrai Hors-Ligne** : Aucune dépendance externe n'est requise pour écrire.
